# Construction ERP - User & Role Management Design Package

Open `index.html` in a browser to view the prototype.

## Contents
- `screens/` contains all HTML screen mockups.
- `assets/styles.css` contains shared UI styling.
- `docs/` contains design brief, screen fields, and Figma designer instructions.

## Scope
This package covers:
- Login
- Forgot Password
- Reset Password
- Dashboard + Sidebar
- Users Listing
- Add/Edit User
- User Details
- Roles Listing
- Create/Edit Role
- Role Details
- Permission Matrix
- Role Hierarchy
- Assign Users to Role
- Approval Workflow Builder
- Activity Logs
