# Client Explanation - Phase 2

Phase 1 covered the foundation of access control: users, roles, permissions, hierarchy, and workflows.

Phase 2 now covers the ERP base setup. These are the master records that will be used by all other modules.

For example:

- HR needs departments and designations.
- Attendance needs projects and sites.
- Mobile attendance needs site geo-fence data.
- Site expenses need expense categories, suppliers, projects, and linked chart of accounts.
- Inventory needs warehouses.
- Accounting needs company VAT/ZATCA settings, suppliers, customers, payables, and receivables.
- Project costing needs project budgets and site setup.

So Phase 2 is the correct next step before designing HR, Accounting, Inventory, Projects, Equipment, and Mobile App modules.
